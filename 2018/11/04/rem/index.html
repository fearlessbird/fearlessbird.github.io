hexo使用篇

发表于 2018-11-04

#模式转换
打开终端：cd hexo文件夹名称
开始调试：hexo s --debug
结束调试：Ctrl+C

创建文章

注：新增文章是在结束调试的模式下

##文本的3种格式

#将文档发布到web上
执行hexo命令：

hexo clean (清空资源文件，可选操作)
hexo g(重新生成资源文件)
hexo server（本地发布文章）
本地发布后查看效果：http://localhost:4000/
hexo deploy(发布文章到github.io)
输入github账号，密码
查看远程效果
